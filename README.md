- Login page
- Auth API
- Landlord API
- Health Check endpoint

## How It Works

- GitHub Actions checks all endpoints every 5 minutes
- Response time data is recorded and committed to this repository
- GitHub Issues are automatically opened when downtime is detected
- Slack notifications are sent on status changes
- Target uptime: **99.5%**
